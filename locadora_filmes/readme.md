# 🎬 Locadora de Filmes - Python

Este projeto simula uma locadora de filmes usando Python.

## 📌 Funcionalidades

- Adicionar filmes
- Editar filmes
- Listar filmes
- Buscar filme por título
- Alugar filmes
- Devolver filmes

## 🧠 Tecnologias usadas

- Python
- Listas
- Dicionários
- Funções
- Menu com while

## ▶️ Como executar

1. Abra o terminal no VS Code
2. Execute:


python main.py


## 📁 Estrutura do projeto

- main.py → menu principal
- filmes.py → funções do sistema
- dados.py → lista de filmes
